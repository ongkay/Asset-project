Cara Install Extension TvLink

  1. Download extension TvLink: Klik Disini
  2. Buka folder download, pindahkan file tvlink ke folder baru, lalu extract file.
  3. Buka Chrome dan buat profil Chrome baru
  4. Ketik chrome://extensions/ di address bar & Enter.
  5. Aktifkan "Mode Developer" di pojok kanan atas.
  6. Klik "Load unpacked" di pojok kiri atas
  7. Pilih folder hasil extrak file ZIP TvLink tadi
  8. Buka extension tvlink dan jika belum login silahkan login.
  1. Download extension TvLink: Klik Disini
  2. Download Install Kiwi Browser dari Play Store: Klik Disini
  3. Buka Kiwi Browser.
  4. Klik ikon titik tiga (menu) di pojok kanan atas.
  5. Pilih menu "Ekstensi" (Extensions) lalu Aktifkan "Mode Developer".
  6. Klik +(from .zip/.crx/.user.js) dan Pilih file tvlink.zip.
  7. Setelah terinstal, klik menu (titik tiga lagi), dan klik "TvLink".
  8. Jika belum login silahkan login
